# Dev AI Pod Weekly Newsletter

Generate the Dev AI Pod weekly newsletter using the @dev-ai-pod-weekly-newsletter agent.

Please invoke the @dev-ai-pod-weekly-newsletter agent to generate a newsletter for the past {{days|7}} days.

The agent will:

- Gather data from Notion, Slack, GitHub, and Linear
- Analyze and summarize the information
- Generate a comprehensive newsletter

Pass the following parameters to the agent:

- days: {{days|7}}
